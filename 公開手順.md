# PWA公開手順（アイコン＋オフライン＋配布対応）

## ✅ 公開済み（2026-07-17）
**本番URL：https://naitou0121.github.io/nukemaki/**
（GitHub Pages・リポジトリ naitou0121/nukemaki・無料・永久）
以下のNetlify手順は不要になった（記録として残す）。更新したい時は index.html 等を直してリポジトリに push すれば数分で反映。

この `PWA配布` フォルダを**ネットに1回上げるだけ**で、
「ホーム画面アイコン・機内モードで動く・データ保持・そのまま配布リンク」が全部揃う。

## いちばんラク：Netlify Drop（アカウント不要でURLが出る）
1. パソコンのブラウザで **https://app.netlify.com/drop** を開く
2. この **`PWA配布` フォルダごと**、ページにドラッグ＆ドロップ
3. 数秒で `https://〇〇〇.netlify.app` というURLが出る ← これが本番URL
   - ※無料アカウントで「Claim（保存）」すると、URLが消えず永続になる（推奨）

## iPhoneに入れる（本命）
1. 出たURLを **iPhoneのSafari**で開く（初回だけ通信が要る）
2. サンプル5冊が表示されればOK
3. Safari下の**共有 → ホーム画面に追加 → 追加**
4. アイコン完成。**以後は機内モードでもアイコンから開ける**（Service Workerがオフライン対応）

## 自分の94冊を入れる（個人データはネットに上げない）
1. 上のアプリをホーム画面から開く
2. `PWA配布` の親フォルダにある **`インポート用データ.txt`** をiPhoneにAirDrop → 開いて全文コピー
3. アプリ下「読み込む」→ 貼り付け → 「この内容を読み込む」→ 94冊入る
   （サンプルは自分のデータで置き換わる）

## 配布
- この同じURLを人に配ればOK。**各自の端末に各自のデータ**が入る（他人には見えない）
- 配布URLに乗るのは白紙/サンプルだけ。貴斗さんの94冊は各自の端末内なので漏れない

## 代わりの手（永続・玄人向け）
GitHub Pages でも可（要GitHubアカウント）。フォルダをリポジトリに push → Pages 有効化。
Netlify で困ったら言って、GitHub 手順を出す。
